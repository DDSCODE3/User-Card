# Random User Generator

## Description
A web application that fetches and displays random user data from an API.

## Features
- Fetch random user information
- Display user details (name, email, phone, location)
- Generate new random users on demand
- Responsive design

## Technologies
- HTML5
- CSS3
- JavaScript (ES6+)
- Random User Generator API

## Installation
1. Clone the repository
2. Open `index.html` in your browser

## Usage
Click the "Generate User" button to fetch a new random user and display their information.

## API
Uses the [Random User Generator API](https://randomuser.me/api/)
